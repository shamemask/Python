Веб страница таблица данных о жителях Средневекого города.
Чтобы запустить сайт необходимо

1. Открыть каталог с сайтом через cmd
2. Выполнить команду: 
venv\scripts\activate
3. Запустить сервер выполнив команду:
flask run 

- Чтобы удалить текущие данные о жителях, необходимо удалить Medieval_city.db
- Чтобы наполнить бд случайными данными о 500 жителях выполнить команду:
python populate.py

- Чтобы подключить сервис к Postgres БД, необходимо изменить настройки
в файле config.py
